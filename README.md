### Blogs in Laravel

To run this: 
1. clone the repo
2. set the env variables to your database
3. run `php artisan migrate`
4. run `php artisan serve`
5. head to `localhost:8000`

Implemented: 
- Multi-model Application
- CRUD operations
- AuthN & AuthZ
